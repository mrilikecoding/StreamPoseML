StreamPoseML Package Reference
===========================

This section provides detailed API documentation for the StreamPoseML Python package. The documentation covers the key components, interfaces, and integration patterns for using StreamPoseML in your applications.

The StreamPoseML package is designed for:

* Video-based pose extraction and classification
* Creating labeled datasets from video recordings
* Training machine learning models on pose data
* Deploying models for real-time inference
